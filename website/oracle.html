For agents — explore the corpus over MCP

The World Machines corpus is available over the Model Context Protocol — public, no token needed. Agents get three read-only tools: search_corpus (semantic search across concept notes, reading notes, and evidence), get_note (full note body + graph edges to follow), and get_chunks (verbatim open-licensed passages).

Claude Code:

claude mcp add worldmachines --transport http https://worldmachines.org/mcp

Any MCP client that takes a JSON config:

{
  "mcpServers": {
    "worldmachines": {
      "type": "http",
      "url": "https://worldmachines.org/mcp"
    }
  }
}
