# Chirag Shah

## Identity

Researcher in information retrieval and human-computer interaction. Co-authored "Agents Are Not Enough" (December 2024) with [[Ryen White]].

## Relevant Work

### "Agents Are Not Enough" (2024, with Ryen White)

Cited in [[Jan 10, 2025 - AI Agents, Machiavelli's Study]]. Key claims:

1. **Broad agent definition**: everything from thermostats to LLM tools qualifies as an "agent" — agents remove agency from users to act on their behalf
2. **The principal-agent problem in AI**: see [[Principal-Agent Problem in AI]]
3. **Current agent failures**: agents fall short on value generation, personalization, and trust
4. **The Sims proposal**: [[Sims (AI User Simulation)]] — user-representing companions that hold intimate knowledge of the user and can represent user interests in agentic ecosystems
