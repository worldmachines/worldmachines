# 1913 Webster's Dictionary

## What It Is

*Webster's Revised Unabridged Dictionary* (1913) is the final major edition of Noah Webster's original American English dictionary before modern abridgement began to strip out etymological richness and literary illustration. It contains approximately 110,000 entries.

## Why It's Preferred

Per [[John McPhee]] via [[James Somers]] (referenced in [[A Good Dictionary]]):
- Definitions are **literary**: they use the word in context, show its range, make it *live*
- Etymology is detailed and informative
- Near-synonyms are distinguished with nuance
- The dictionary rewards browsing, not just lookup

Modern dictionaries optimize for disambiguation in seconds. The 1913 Webster's optimizes for a *writer's* encounter with language — generative, slow, exploratory.

This is what [[Lexicographic Richness]] looks like in practice.

## Installation on macOS

From [[A Good Dictionary]] (updated instructions):
1. Download `websters-1913.dictionary.zip` from the GitHub Releases page for WebsterParser
2. Open Dictionary app → File > Open Dictionaries Folder (or navigate to `~/Library/Dictionaries`)
3. Move the unzipped `.dictionary` file into that folder
4. Restart Dictionary app, open Settings (⌘,), enable Webster's Unabridged Dictionary (1913)

Also available online; the OED is the expensive authoritative alternative.
